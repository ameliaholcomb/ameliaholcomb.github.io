AngularJS DialogWindows Module

The goal of this project was to create a standalone module for fully functional dialog windows using angularjs. The windows themselves behave as application windows do, with a draggable (heading?) bar that includes buttons for minimization, maximization, and permanent close. The windows themselves can also be resized and reorder automatically according to last click.

The windows can be filled with arbitrary HTML/CSS/Javascript content supplied by the server, and their state (position, size, minimization) is stored on the server and therefore easily persisted across logins. Minimal information is transmitted from the server in order to retrieve that state.
